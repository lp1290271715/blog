# arguments

保存实参的数组，会跟传入的形参形成映射关系“你变我也变”

function test(a,b){
    test.length //2
    arguments.length // 3
    a = 5
    arguments[0] // 5
    arguments[1] = 6
    b //6
}
test(1,3,4)
